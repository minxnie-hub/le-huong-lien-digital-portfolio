## Chạy thử tại máy

Không cần cài package. Mở `index.html`, hoặc chạy server cục bộ:

```bash
python -m http.server 8080
```

Sau đó mở `http://localhost:8080`.

## Deploy GitHub Pages

1. Tạo repository và đẩy toàn bộ thư mục này lên nhánh `main`.
2. Vào **Settings → Pages**.
3. Ở **Build and deployment**, chọn **GitHub Actions**.
4. Workflow `.github/workflows/deploy.yml` sẽ tự triển khai website.
